# 📊 电商平台用户满意度影响因素分析

## 📌 项目简介

本项目基于 Olist 电商公开数据集，从订单层面出发，分析影响用户评分（用户满意度）的关键因素，并结合数据分析结果提出业务优化建议。

在电商业务中，GMV 的长期增长依赖于用户体验与复购行为，而用户评分是衡量满意度的核心指标。因此，本项目重点关注：

> 哪些因素在实际业务中真正影响用户评分？

---

## 🧠 核心结论

通过数据分析与建模，得到以下关键结论：

- 🚚 **配送是否延迟是最关键因素**  
  一旦订单延迟，用户评分显著下降

- ⏱️ **延迟时间越长，负面影响越强**  
  用户对“迟到”高度敏感

- 📦 **提前送达的正向作用有限**  
  “不迟到”比“更快”更重要

- 💰 **运费对评分存在负向影响**  
  运费越高，用户满意度越低

- 统一分析粒度（以 order_id 为核心）

👉 产出：订单级分析数据表

---

### 3️⃣ 探索性分析（Power BI）

- 构建 `order_summary` 表（DAX）
- 对关键变量进行分箱处理
- 分析变量与评分之间的关系：

  - 配送延迟 vs 评分  
  - 订单金额 vs 评分  
  - 运费占比 vs 评分  

👉 产出：核心变量筛选结果
